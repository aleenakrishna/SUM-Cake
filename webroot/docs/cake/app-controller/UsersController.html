method register [line 232]

void register( )

Register

Allows users to register. This will send a welcome email (if enabled) allowing them to activate their account from a link in an email. This provides a "double opt-in" security measure so people cannot be given access on an email account they do not own. If autoValidate is set to true, the account will be activated automatically.
